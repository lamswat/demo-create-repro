# 1. Đăng nhập 
* Đăng nhập vào MariaDB Server với lệnh mysql -u root -p và nhập mật khẩu root.
                mysql -u root -p 
* Trong đó: root là user đăng nhập, bạn thay bằng tài khoản của bạn. Tùy chọn -p để bạn nhập mật khẩu.
